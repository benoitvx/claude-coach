# Vélo indoor — Devenir un cycliste complet (9 sem, Confirmé)

> Référence d'inspiration (Decathlon Coach). Surtout une **banque de séances HT
> de ~40-58 min** (compatible créneau home trainer ≤ 60-70 min), classables par
> filière. Zones 1-5 → zone 5 = PMA/VO2max, zone 3 = seuil bas, zone 4 = seuil
> critique. Recalibrer sur FTP/zones de l'athlète.

## Filières travaillées (banque réutilisable)

### PMA / VO2max (zone 5)
- **30/30** : 10×(30" Z5 / 30" Z2) — base ; monte à 2×8 puis efforts longs.
- **15/15** : 2×[8×(15" Z5 / 15" Z2)] récup 8' entre séries.
- **Efforts longs** : 4×(1' Z5/1' Z2) + 3×(1'30 Z5/1'30 Z2) + 2×(2' Z5/2' Z2), 100 rpm.

### Force maximale / sprint
- **Sprints courts** : 5×(7" sprint max / 5' récup à 100 rpm), braquet lourd.
- **Pyramide** : 6 sprints 15"/20"/25"/20"/15", 60-70 rpm, récup 5'.
- **Force sous-max** : 2×[5×(30" effort / 1' récup)] à ~90 % FCmax, 60-70 rpm.

### Endurance de force (type montée)
- 3×(5' soutenu / 3' récup) à ~90 % FCmax, braquet important.
- 2×(10' soutenu / 5' récup), 110 rpm en récup.

### Seuil anaérobie (zone 3/4)
- Cadence variable : 3×(5' Z3 @80 rpm) puis @95 rpm puis @110 rpm, récup 3'.
- Blocs longs : 2×(10' Z3 @80 rpm / 5') puis 2×(10' Z3 @110 rpm).
- Alternance Z3/Z4 : 4×(5' de 30" Z3 / 30" Z4 / 2' récup).

### Lactique (anaérobie)
- 2×[6×(15" max / 30" récup active)] récup 10' entre séries (récup raccourcie au fil des semaines : 30" → 15").

### Technique de pédalage (gammes)
- Unijambiste : 5×(1' jambe D / 15" 2 jambes / 1' jambe G / 1'30 récup).
- Combiné : 5×(1' D / 1' G / 1' tirage / 2' vélocité / 1' récup).
- Vélocité : 2×[5×(1' vélocité / 1' récup souple)].

## Périodisation (9 sem)
S1-2 PMA + force + sprint · S3-4 force endurance + PMA + technique · S5-6 seuil +
technique · S7-8 seuil + force + lactique · S9 affinage seuil + vélocité.
Micro-coupure 3-4 j conseillée entre S4 et S5. Toute séance : éch progressif
10-15' + RAC 10' + étirements 15'.
