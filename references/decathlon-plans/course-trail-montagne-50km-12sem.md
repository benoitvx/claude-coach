# Trail montagne — finir en 12 semaines (Confirmé)

> Référence d'inspiration (Decathlon Coach). À adapter, pas à appliquer tel quel.
> Le plan suppose footing en % FC max ; pour cet athlète, raisonner en zones
> calibrées sur `athlete show` (FCrepos/FCmax).

**Structure** : 3 cycles de 3 semaines + semaines de régénération.
Cycle 1 (S1-3) · Régé (S4) · Cycle 2 (S5-7) · Régé (S8) · Cycle 3 (S9-11) · Régé + course (S12).
Trois piliers récurrents : **endurance vallonnée** (70-75 % FC), **fractionné en côtes**, **rando-course** (sortie longue alterne marche en côte / course facile).

## Détail séance par séance

### S1 — Première sortie longue
- Endurance vallonnée 1h30 (70 % FC, max 75 %) — footing nature, surfaces alternées
- Endurance vallonnée 1h30 (70 % FC) — hydratation 1 gorgée/20 min
- Endurance vallonnée 2h (65-75 % FC)

### S2 — Première rando-course
- Endurance vallonnée 2h (65-75 % FC)
- **Côtes** ~55 min : éch 30' + 4×6' en côte (pente 5-7 %, récup descente ~4') + RAC 10'
- **Rando-course 3h** : marche (côtes) / course facile (plat-descente), test matériel + hydratation

### S3 — Augmentation des distances
- Endurance vallonnée 1h30 (70-75 % FC)
- **Côtes** ~60 min : éch 20' + 3×8' en côte (récup ~5' descente) + RAC 10'
- **Rando-course 4h**

### S4 — Régénération
- Endurance plat 1h (70-75 % FC)
- Endurance vallonnée 1h (70-75 % FC)
- Repos total

### S5 — Priorité à la durée
- Endurance vallonnée 1h30 (70-75 % FC)
- **Côtes progressif** ~60 min : éch 20' + 10' côte (récup ~6'30) + 8' côte (récup ~5') + 6' côte (récup ~4') + RAC 10'
- **Rando-course 5h** : terrain proche course, test ravito

### S6 — Priorité au dynamisme
- Endurance plat 1h (70-75 % FC)
- **Côtes** ~60 min : éch 30' + 3×6' côte (récup 4' descente) + RAC 10'
- Endurance vallonnée 1h30 (70-75 % FC)

### S7 — Travail de la durée
- Endurance vallonnée 1h30 (70-75 % FC)
- **Côtes** ~60 min : éch 30' + 3×6' côte + RAC 10'
- **Rando-course 6h** (sortie la plus longue) : test équipement/ravito

### S8 — Régénération
- Endurance 45 min (70-75 % FC)
- Endurance vallonnée 1h (70-75 % FC)
- Repos total

### S9 — Dernier travail sur la durée
- Endurance vallonnée 2h (65-75 % FC)
- **Côtes** ~65 min : éch 20' + 3×10' côte (récup 6'30 descente) + RAC 10'
- **Rando-course 4h**

### S10 — Dernier travail dynamique
- Endurance plat 1h (70-75 % FC)
- **Côtes** ~55 min : éch 20' + 3×6' côte + RAC 10'
- Endurance vallonnée 2h (65-75 % FC)

### S11 — Diminution progressive (mar/jeu/sam)
- Endurance plat 1h · Endurance vallonnée 1h15 · Endurance plat 1h (70-75 % FC)

### S12 — Affûtage + compétition
- Endurance 30 min (70-75 % FC, réductible)
- Repos total
- **Compétition** : éch 20', départ contrôlé, accélération possible derniers km

## Idées à retenir pour la périodisation
- Montée en durée de la sortie longue : 3h → 4h → 5h → 6h sur les 3 cycles.
- Le D+ se travaille via les côtes (volume de côte qui monte 4×6' → 3×10') + rando-course en terrain spécifique.
- Affûtage net sur les 2 dernières semaines.
