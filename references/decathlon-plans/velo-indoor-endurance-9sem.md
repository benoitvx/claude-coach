# Vélo indoor — Progresser en endurance (9 sem, Intermédiaire)

> Référence d'inspiration (Decathlon Coach). Paradoxe assumé par le coach :
> construire de l'endurance sur HT par sessions ≤ 1h n'est pas évident → le plan
> mise sur **technique de pédalage + seuil + force endurance** pour densifier des
> séances courtes. Utile quand le créneau home trainer est limité (≤ 60-70 min).
> Recalibrer zones sur FTP/FC de l'athlète.

## Blocs types (~40-55 min)
- **Technique pédalage** : unijambiste, tirage, vélocité 115-130 rpm
  (ex. 5×(2×1'/1'), 5×(1' D/1' G/1' tirage/2' vélocité/1')).
- **Force maximale** : 5×(7" sprint / 5' récup) ; pyramide 15-25".
- **Force endurance** : 3×(5' soutenu / 3') ; 2×(10' soutenu / 5'), ~90 % FCmax.
- **Seuil (Z3/Z4)** : 4×(5' de 30" Z3/30" Z4 / 2') ; 2×(10' Z3 @80 puis @110 rpm).
- **PMA (Z5)** : 30/30 (10×) puis 2×8×(30/30) ; 15/15 (2×8).

## Périodisation (9 sem)
- **S1-3 — fondations** : technique + force max + premier seuil.
- **S4-5 — endurance de force** (micro-coupure 3-4 j avant S5).
- **S6-8 — intensité** : introduction PMA (30/30 puis 15/15), seuil à cadences variées.
- **S9 — consolidation** : technique + seuil alterné Z3/Z4.

## Idée à retenir
- Pour de l'endurance « utile » en sessions courtes : empiler seuil + force
  endurance + vélocité plutôt que du Z2 long (réservé aux sorties extérieures).
- Cohérent avec la contrainte matériel : home trainer = qualité courte ;
  l'endurance pure longue se fait dehors quand la météo le permet.
